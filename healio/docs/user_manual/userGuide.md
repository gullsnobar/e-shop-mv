# HEALIO User Guide

## Getting Started
1. Download HEALIO from App Store / Play Store
2. Register with email or sign in with Google
3. Complete profile setup (optional)

## Key Features

### Medication Management
- Tap '+' on Medications tab to add a new medication
- Set dosage, frequency, and reminder times
- Mark doses as taken when reminded
- View adherence charts on Dashboard

### Fitness Tracking
- Connect Google Fit for automatic data sync
- Manually log steps, sleep, and vitals
- Track water intake with quick-add buttons
- Log meals for calorie tracking

### AI Health Assistant
- Open AI Chat tab
- Ask health-related questions
- Get personalized recommendations
- Request lab report analysis

### Managing Appointments
- Add doctor appointments with location
- Receive day-before reminders
- Mark appointments as completed

### Sharing with Trusted Contacts
- Add family members or caregivers
- Choose what data to share
- Contacts receive email verification
