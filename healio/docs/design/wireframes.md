# UI Wireframes & Design Guide

## Color Palette
- Primary: #4A90D9
- Secondary: #50C878
- Accent: #FF6B6B
- Background: #F5F7FA
- Surface: #FFFFFF
- Text Primary: #333333
- Text Secondary: #666666

## Typography
- Headings: Roboto Bold
- Body: Roboto Regular
- Accent: Roboto Medium

## Screen Layout (Bottom Tabs)
1. **Home** - Dashboard with health score, quick stats, upcoming meds/appointments
2. **Dashboard** - Detailed charts and analytics
3. **Medications** - Medication list, add/edit, adherence tracking
4. **Fitness** - Steps, sleep, heart rate, water, diet
5. **AI Chat** - AI health assistant chatbot

## Component Style Guide
- Card border-radius: 12px
- Button border-radius: 8px
- Spacing unit: 4px
- Shadow: 0 2px 8px rgba(0,0,0,0.1)
